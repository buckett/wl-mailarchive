Enviar email para os participantes do site via Arquivo do email

Para enviar email para todos os membros do seu site de curso ou projeto:

  1. No menubar do seu site de curso ou projeto, clique em Arquivo do email. Se você não consegue ver o Arquivo do email, o proprietário do site (ou alguém com as permissões adequadas) deve adicioná-lo ao site, veja Adicionar Arquivo do email no seu site.

  2. Na tela do Arquivo do email, em "E-mail enviado para os seguintes endereços serão arquivados e enviados aos participantes", você verá o endereço de email do seu site. Acima dele, você verá uma mensagem informando se você está autorizado a enviar e-mail. Se este site está configurado para aceitar email somente de participantes do site (veja Personalizar um Arquivo do email), você verá também o endereço de onde você está autorizado a enviar e-mail, use essa conta para enviar mensagens para o endereço do site. As mensagens irão para todos os participantes do seu site de curso ou projeto, e serão arquivados on-line no Arquivo de e-mail.

Nota: Você não pode usar o Arquivo do E-mail para enviar e-mail para determinados indivíduos. As mensagens enviadas para o endereço do seu site de e-mail serão recebidas por todos os participantes do site. Para enviar mensagens para os participantes individuais, use a ferramenta Mensagens.
